# Knowledge Base — Salons & Spas

This directory contains industry context that AI assistants reference when running skills. It helps the AI use correct terminology, follow industry norms, and produce output that sounds like it was written by a salons & spas professional.

## Contents

| Directory | What's in it |
|-----------|-------------|
| `industry-overview.md` | Market size, trends, common pain points, and where AI fits |
| `terminology/` | Industry jargon, acronyms, and definitions |
| `regulations/` | Compliance requirements, licensing, safety standards |
| `best-practices/` | Industry standards and recommended approaches |
| `tools-ecosystem/` | Common software, platforms, and tools used in salons & spas |

## How it's used

When a skill runs, the AI loads relevant knowledge base files to:

1. Use correct industry terminology instead of generic language
2. Reference applicable regulations and compliance requirements
3. Follow industry best practices in generated output
4. Understand the tools and software the business likely uses

## Contributing

If you notice incorrect terminology, outdated regulations, or missing best practices, please open a PR. Industry professionals' input makes these skills better for everyone.
